# Gate Futures Bot

Автоматический торговый бот для фьючерсных листингов на бирже Gate.io.

## Структура проекта

```
Robot_Gate/
├── src/
│   ├── api/           # API клиенты (gate_client, monitoring, websocket_client)
│   ├── bot/           # Ядро торгового бота (core.py)
│   ├── db/            # Модели, подключение к БД, настройки
│   ├── risk/          # Риск-менеджмент (balance, rate limiter, circuit breaker)
│   ├── telegram/      # Telegram бот (уведомления и управление)
│   ├── trading/       # Торговый модуль (trader.py)
│   └── utils/         # Утилиты и конфигурация
├── tests/             # Тесты
├── logs/              # Логи
├── config/            # Конфигурационные файлы (config.yaml)
├── main.py            # Точка входа
├── init_db.py         # Инициализация БД
└── requirements.txt   # Зависимости
```

## Установка

### 1. Клонирование и установка зависимостей

```bash
# Создаем виртуальное окружение
python -m venv venv

# Активируем (Linux/Mac)
source venv/bin/activate

# Активируем (Windows)
venv\Scripts\activate

# Устанавливаем зависимости
pip install -r requirements.txt
```

### 2. Настройка конфигурации

```bash
# Копируем шаблон .env
cp .env.example .env

# Редактируем .env
nano .env
```

Обязательно заполните:
- `GATE_API_KEY` - API ключ Gate.io
- `GATE_API_SECRET` - API секрет Gate.io
- `DB_HOST`, `DB_NAME`, `DB_USER`, `DB_PASSWORD` - параметры PostgreSQL
- `TELEGRAM_BOT_TOKEN` - токен Telegram бота
- `TELEGRAM_ADMIN_ID` - ваш Telegram ID

### 3. Настройка базы данных

```bash
# Создаем базу данных PostgreSQL
createdb gate_bot

# Инициализируем таблицы
python init_db.py
```

## Стратегия торговли

**Стратегия: SHORT на новых фьючерсных листингах**

**Вход в позицию (SHORT):**
- Обнаружение нового листинга через мониторинг API
- Ожидание заполнения стакана маркетмейкером (min 5 уровней, спред <= 1%)
- Проверка минимальной ликвидности на лучшем уровне
- Проверка риска перед входом
- Плечо: 20x, размер позиции: $10

**Усреднения (при росте цены):**
- Уровень 1: +300% от цены входа
- Уровень 2: +700% от цены входа
- Уровень 3: +1000% от цены входа
- Максимум 3 усреднения

**Выход:**
- Take Profit: 2% от средней цены входа
- Stop Loss: отключен (0%)
- Timeout: 720 часов (30 дней)

**Защита:**
- Дневной лимит потерь: $1000
- Минимальный баланс: $100
- Rate limiting: 10 запросов/сек
- Circuit breaker: 3 ошибки → блокировка на 60 сек
- Защита баланса: при просадке 50% — перевод 25% со спота на фьючерсы

## Запуск

```bash

source venv/bin/activate  # Linux/Mac
```

1. Бот валидирует конфигурацию
2. Инициализирует базу данных
