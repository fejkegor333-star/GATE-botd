# КОММЕРЧЕСКОЕ ПРЕДЛОЖЕНИЕ
## Автоматический торговый бот для фьючерсных листингов Gate.io

---

### 📋 ОБЩАЯ ИНФОРМАЦИЯ

**Проект:** Разработка торгового бота для автоматической торговли новыми фьючерсными контрактами USDT на бирже Gate.io

**Стратегия:** SHORT (шорт) по новым листингам с фиксацией прибыли и сеткой усреднений

**Стоимость:** **40 000 руб.**
- Основной бот: 35 000 руб.
- Telegram-панель управления: 5 000 руб.

**Срок разработки:** **2-3 недели**

---

### 🎯 ФУНКЦИОНАЛ БОТА

#### 1. Автоматический поиск новых листингов
- Мониторинг фьючерсов Gate.io 24/7
- Определение новых контрактов в реальном времени
- Анализ заполнения стакана маркетмейкером
- Автоматический вход при соблюдении условий

#### 2. Интеллектуальная торговля
- **Фиксация прибыли каждые 2%** движения цены вниз с автоматическим переоткрытием позиции
- **Сетка усреднений** на уровнях 300%, 700%, 1000% от цены входа
- **Фиксированная сумма усреднений** в USDT (настраивается)
- **Учет истории усреднений** — при изменении настроек бот помнит уже совершенные усреднения
- **Бесконечный шорт** — без стоп-лосса, стратегия переоткрытия позиций

#### 3. Умные фильтры входа
- **ATH ratio > 0.3** — работа только с монетами, упавшими с исторического максимума
- **30 дней с листинга** — автоматическое завершение работы с монетой через заданный период
- **Анализ ликвидности** — вход только при достаточном объеме в стакане

#### 4. Защита капитала
- **Автоматический чекер баланса** — при просадке 50% бот автоматически переводит средства со спота на фьючерсы для предотвращения ликвидации
- **Ограничение количества позиций** — настраивается через Telegram
- **Режим тестирования** — безопасный режим для отладки с минимальными объемами

#### 5. Telegram-панель управления
Управление всеми параметрами бота прямо из мессенджера:

**Команды:**
- `/status` — текущие позиции и P&L
- `/settings` — просмотр и изменение настроек
- `/balance` — балансы спот/фьючерсы
- `/positions` — детальная информация по позициям
- `/stop` — остановка торговли (новые позиции не открываются)

**Настраиваемые параметры:**
- Количество одновременно работающих монет (0 = остановка)
- Начальный объем позиции в USDT
- Количество усреднений (максимум)
- Сумма усреднения в USDT
- Уровни усреднений в процентах
- Процент тейк-профита
- ATH ratio порог
- Количество дней с листинга

#### 6. Уведомления в Telegram
- 🟢 Открытие позиции с деталями
- 🔴 Закрытие позиции с P&L
- 📊 Усреднения с указанием уровня и новой цены
- 💰 Балансовые операции (защита от ликвидации)
- ⚠️ Критические события и ошибки
